## 如何使用

將此目錄加入 Claude Code 的技能搜尋路徑，即可使用這些技能。

## 如何新增技能

1. 建立一個以技能名稱命名的目錄
2. 在目錄中建立 `SKILL.md` 檔案

### SKILL.md 格式

```markdown
---
name: 技能名稱
description: 技能的簡短說明，用於觸發判斷
---

# 技能標題

## 說明

詳細說明這個技能的用途和行為。

## 行為指引

1. 步驟一
2. 步驟二

## 範例

展示使用情境的範例。
```

### 欄位說明

- **name**: 技能的唯一識別名稱
- **description**: 簡短說明，Claude 會根據此判斷何時觸發技能
